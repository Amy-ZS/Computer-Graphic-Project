# GPU Fluid Simulation – Final Project

**Group 7:** Amy Shi, Larry He, Qinan Shen  
**Course:** CS 428/523 – Computer Graphics  

---

## Project Description

A real-time fluid simulation using Smoothed Particle Hydrodynamics (SPH) running on GPU compute shaders. Fly through a forest environment, watch water particles move with realistic pressure and viscosity, and press F to render the water surface as a mesh.

---

## How to Run

1. Download `FluidSim_Final.zip` from the submission
2. Extract the zip file to a folder
3. Double-click **`Start_FluidSim.exe`**
4. Click "Start" and play!

No installation needed. Just extract and run.

---

## Controls

| Key | Action |
|-----|--------|
| W/A/S/D | Fly around |
| E | Move up |
| Q | Move down |
| Mouse | Look around |
| ESC | Pause / unlock mouse |
| F | Pause time + render water mesh |

---

## Game Flow

1. **Start Menu** → Click "Start"
2. **Play** → Fly around, watch water particles fall
3. **Press F** → Pauses, renders water surface from particles
4. **Press F again** → Resumes, particles come back
5. **End Menu** → Restart or Quit

---

## Known Issues

- Mesh render (F key) can be slow on laptops
